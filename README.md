# Журнал

Программа представляет собой простой консольный журнал для хранения данных о студентах и их оценках
Пользователь может добавлять новые записи, просматривать список всех студентов и выполнять фильтрацию по среднему баллу

---

## Используемые пакеты

В начале подключаются стандартные пакеты Go:

* **bufio** - построчное чтение данных из консоли
* **fmt** - форматированный ввод/вывод
* **os** - работа со стандартным вводом
* **strconv** - преобразование строк в числа
* **strings** - обработка строк

---

## Структура данных

```go
type Student struct {
    Name   string
    Marks  []int
    Medium float64
}
```

Каждый студент хранит:

* **Name** - фамилия и имя
* **Marks** - список полученных оценок
* **Medium** - итоговый средний балл

---

## Расчёт среднего балла

У структуры есть метод `calc()`
Он подсчитывает сумму всех оценок и делит её на количество элементов
Если оценки отсутствуют средний балл становится равным нулю

---

## Добавление студента

Функция обработки добавления:

1. Пользователь вводит ФИО
2. Проверяется, существует ли студент с таким ФИО
3. Далее вводятся оценки. Пустая строка завершает ввод
4. Каждая оценка проверяется:
   * является ли числом,
   * входит ли в диапазон **1–5**.
5. После ввода создаётся структура студента и помещается в таблицу
6. Средний балл вычисляется автоматически

---

## Фильтрация по среднему баллу

Есть возможность показать всех студентов, чей средний балл ниже заданного значения
Вводится число, затем проходит поиск по всей базе

---

## Вывод информации

Программа умеет:

* выводить данные одного студента,
* показывать сразу всех,
* отображать список команд (`help`).

---

## Основной цикл программы

После запуска работает бесконечный цикл, ожидающий команды пользователя:

* **add** - добавить нового студента
* **list** - показать всех
* **filter** - отобрать студентов по среднему
* **help** - список всех команд
* **exit** - завершить программу
